
# 🌙 Nyxa ASM Playground

> “The night runs in assembly.”

Experimental x86_64 assembly playground focused on low-level programming, CPU architecture, memory exploration, and systems engineering.

---

## 💫 About

Nyxa ASM Playground is the first step of the Nyxa ecosystem.

This repository exists as a learning laboratory for:

- x86_64 Assembly
- Linux syscalls
- CPU architecture
- memory manipulation
- stack internals
- low-level debugging
- systems programming fundamentals

The goal is to understand the machine beneath the abstraction.

---

## 🧠 Philosophy

Nyxa is not focused on speedrunning operating system development.

The purpose is to deeply understand:

- how software becomes machine code
- how CPUs execute instructions
- how memory behaves
- how operating systems emerge from hardware

Closer to the machine.
